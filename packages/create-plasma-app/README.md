# create-plasma-fhir-app
CLI for generating a PlasmaFHIR app. This is the recommended way to create a new PlasmaFHIR application.

## Usage
`npx create-plasma-app`

## Development
- `yarn start` to run/test it

Also test with...
- `npm i -g`
- `npx create-plasma-app`

## Deployment

### Publish/Deploy
- Login as `plasmafhir`
- Increment version
- `npm publish`
